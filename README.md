## msVKMarket

Экспорт данных из минишопа в контачевский маркет

 - начало 05.08.2018

##### Процес
    "+" - готово    
    "-" - не готово
    "~" - сейчас в процессе
 - `+` Группы
    - `+` cоздание
    - `+` редактирование
    - `+` удалени
 - `~` Подборки
    - `+` создание
        - `-` загрузка обложки
    - `+` редактирование
    - `+` удалени
    - `~` экспорт 
 - `-` Товары
    - `-` создание 
    - `-` редактирование 
    - `-` удаление
    - `-` вывод
        - `-` просмотры из разных групп
        - `-` лайки из разных групп
        - `-` комментарии из разных групп
    - `-` плагин на
        - `-` опубликован
        - `-` удален
